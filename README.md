# Navegación entre páginas con Flutter
# Zoé Aranza Alva Vazquez 6°J

## Mi prompt o pregunta IA

Lenguaje Dart Flutter, nivel principiante. Navegación entre 3 páginas utilizando rutas nombradas. Desde main llamar a la página 1, en AppBar de la página 1 mostrar el texto "Inicio Alva Vazquez 6°J" en color ámbar, fondo negro e íconos blancos. En body agregar un contenedor redondeado color naranja pastel (200 x 200) con texto negro y centrado además dde un botón de color negro con texto naranja pastel que redirija a la página 2.
En la página 2 un appbar con texto rosa mexicano ("Segunda Págia 6°J"), fondo negro, íconos blancos. En body agregar una imagen desde la red y un botón para avanzar a la página 3.
En la página 3 una AppBar con el texto "Página 3".
Todo en un solo archivo.
